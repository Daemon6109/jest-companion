# TestEZ Companion

[![License][license-shield]][license-url]
[![Discord Server][discord-shield]][discord-url]
[![Visual Studio Marketplace][marketplace-shield]][marketplace-url]

TestEZ Companion is a Visual Studio Code extension that enables Roblox developers to be able to preview their [TestEZ](https://roblox.github.io/testez/) test results inside VS Code.

To contribute to the VS Code extension, go to the [extension](extension) directory.
To contribute to the Roblox Studio plugin, go to the [plugin](plugin) directory. Joining the Discord server is encouraged!
